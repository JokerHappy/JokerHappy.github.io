iOS 9 Universal Links

发表于 2016-04-10 | 分类于 iOS

Apple 推出Universal Links (通用链接)：一种能够方便的通过传统 HTTP 链接来启动 APP, 使用相同的网址打开网站和 APP。

通过唯一的网址, 不需要特别的schema就可以链接一个特定的视图到APP 里面 。比如：在知乎中使用了通用链接, 那么用户在Safari、UIWebView或者 WKWebView点击一个链接, iOS设备上的知乎app会在知乎里面自动打开这个页面, 如果没有安装则在Safrai中打开响应链接。
